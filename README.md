# 💰 FinTrack

### A Modern Personal Finance Dashboard  
Built with **React 18 · TypeScript · Recharts · Recoil · Tailwind CSS**

---

## 📖 About

**FinTrack** is a feature-rich personal finance dashboard that helps users track income, expenses, and savings through interactive visualizations and intelligent insights.

This project was built with a **product-first approach**:
- Designed UI in **Figma before coding**
- Defined **data model first**
- Focused on **real-world UX patterns**

---

## ✨ Features

### 📊 Dashboard Overview
- Total balance, income, and expenses cards  
- Month-over-month change indicators  
- Interactive charts for trends  

### 💳 Transactions
- Paginated and sortable table  
- Mobile-friendly card view  
- Role-based actions (Admin vs Viewer)  

### 🧠 Smart Insights
- Financial health score  
- Spending pattern detection  
- Savings analysis with recommendations  

### ⚙️ Extras
- Dark mode with persistence  
- LocalStorage-based data layer  
- Fully responsive UI
